# v2.3.2_20260301（zhengchen-cam）版本说明

发布日期：2026-03-01

分区体系：`v2`

## 本次新增
1. 本地闹钟能力增强：支持设置、查询、启停、删除、停止、贪睡（snooze）。
2. 本地倒计时能力增强：支持设置、查询、取消、停止，到点触发本地提醒。
3. 倒计时 UI 交互：倒计时过程动态显示 `HH:MM:SS`，进入最后 60 秒切换大字号秒数倒计时。
4. 本地 MCP 音乐能力：支持 URL 音频播放（`wav/mp3/ogg`），并补充歌单推送、检索与播放相关工具。
5. 自定义唤醒词流程增强：当前构建包含 `hi dobi`。

## 本次修复
1. 修复闹钟/倒计时到点后无法通过按键终止的问题。
2. 修复提醒停止后的状态回退逻辑：按一次唤醒键即可停止提醒并返回待机界面。
3. 查询闹钟时默认不返回已关闭（禁用）闹钟，避免无效结果干扰。

## 音频与提醒
- 到点提醒音：`main/assets/common/naozhong1.mp3`
- 倒计时临近滴答音：`main/assets/common/countdown_tick.ogg`
